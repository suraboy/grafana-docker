# Grafana Docker
using for run test grafana.

Get Started
-----------

Run the docker for development:
---------------------
You can start the application and run the containers in the background :

```bash
docker-compose up -d
```
```bash
docker-compose down
```

Running Application
------------------------------------
Open the browser
```bash
http://localhost:3000
```
